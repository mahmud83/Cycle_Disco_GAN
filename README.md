# Cycle_Disco_GAN
Artificial Intelligence GAN

## Description
CycleGAN vs DiscoGAN
* 동일데이터 셋에 대해서 결과비교
```bash
CycleGAN dataset -> DiscoGAN
DiscoGAN dataset -> CycleGAN
```
* Geometry 변경이 가능한 CycleGAN 만들기 (alpha)
```bash
deformable CNN + CycleGAN
```
